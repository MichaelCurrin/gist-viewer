Gist Viewer

A live list of all my public gists

When viewing multiple pages of my gists directly on GitHub itself, that becomes impractical. A one-page solution keeps thing easy to manage.

About

This project is open-sourced. It is easy to fork and deploy using your own username. To get your own site setup like this one, just follow the instructions on the repo's README.md.

If you want to contribute to make this project better, issues and pull requests are welcome. See the Contributing doc.
